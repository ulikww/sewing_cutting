Общие положения

  1. Припуски по талии 1-1,5 см
  2. Припуск по низу 2 см
  3. Припуск по ширине пояса 2,5 см
  4. Длина пояса равна обхват талии + 1,5-2см
  5. Длина резинки равна обхват талии - 1/3 полуобхвата талии и плюс 1,5 см припуск на шов.

Порядок действий по пошиву солнца

  • Кроим на ткани юбку
    3. Размечаем радиус на юбке по низу
    4. Размечаем радиус талии
    5. Вырезаем юбку
  • Работа с резинкой
  • Работа на машинке
    1. Сметываем оверлоком боковые швы
    3. Сметка пояса в цельный
    4. Вставка резинки в пояс
    5. Скалываем пояс с юбкой
    6. Сметка на оверлоке пояса с юбкой
  • Работа с низом изделия
    1. Раскладка на столе для выверки низа изделия
    2. Еще два способа отметить низ изделия
    3. Отмечаем линию подгиба, 1,5 см либо 2 см
    4. Обрабатываем низ на распашивальной машинке ( в домашних условиях можно обметать оверлоком, потом сделать строчку двойной иглой за отсутствием распашивальной)
    5. После того, как шов проложен на распашивальной машине, обрезаем под шов лишнюю ткань подгиба
    6. Можно дополнительно проложить шов на распашивальной машинке по поясу, чтобы резинка внутри не "ходила". Делается это в одну иглу и один петлитель
